Quick Reference

üìå ESSENTIAL FUNCTIONS
  • imread() - read images
  • imshow() - display images
  • im2double() - convert to [0,1]
  • rgb2gray() - color to grayscale
üìä HISTOGRAM TOOLS
  • imhist() - compute histogram
  • histeq() - equalize/match
  • cumsum() - CDF computation
  • bar() - plot histogram
üîç FILTERING
  • imfilter() - apply filter
  • fspecial() - create kernels
  • ones()/zeros() - manual kernels
‚ö†Ô∏è Quick Reminders
imshow(I, [])

Auto-scales display ‚Äî use when unsure of range!

double() vs im2double()

double() keeps [0,255], im2double() normalizes to [0,1]

Filter normalization

Always divide filter by sum of coefficients!
