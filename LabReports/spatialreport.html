All lab work must be submitted in a Word document (.docx) with a proper UEL cover page at the beginning. Use only textbook images from Gonzalez and Woods, 3rd edition throughout your experiments and always cite the figure name (e.g., Fig0338(a)(blurry_moon).tif) in your report.

Document structure

Required: UEL cover page, then each task in order with task statement copied first, followed by your MATLAB code, figures, and written observations.

Image policy

Mandatory: Use only Gonzalez and Woods textbook images (3rd edition). State the image filename clearly in each task section of your report.

Submission format

File type: Microsoft Word (.docx) only. Include all code, figures, and explanations inside the document. No separate script files.

Document template: Each task section should have: (1) Task title and full statement, (2) MATLAB code blocks with clear comments, (3) Result figures with captions, (4) Written deliverable answering the specific questions or observations required by that task.
